## ✨ Features

- 🔐 **Login Screen** — Clean email & password input with form validation
- 📝 **Sign Up Screen** — User registration form with a polished UI
- 🎨 **Modern UI** — Styled with HTML & CSS for a sleek, responsive look
- 📦 **Native Android Wrapper** — Java-based Android app rendering the web UI
- ⚡ **Lightweight** — Minimal dependencies, fast to build and run

## 🚀 Getting Started

### Prerequisites

- [Android Studio](https://developer.android.com/studio) (latest stable version recommended)
- Android SDK (API Level 21+)
- Java Development Kit (JDK 8 or higher)

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/maliksalimov/signup-login-page.git
   ```

2. **Open in Android Studio**
   - Launch Android Studio
   - Select **File → Open**
   - Navigate to the cloned project folder and click **OK**

3. **Sync Gradle**
   - Android Studio will automatically prompt you to sync Gradle
   - Click **Sync Now** and wait for the build to complete

4. **Run the application**
   - Connect an Android device or start an emulator
   - Click the **Run ▶** button or press `Shift + F10`
